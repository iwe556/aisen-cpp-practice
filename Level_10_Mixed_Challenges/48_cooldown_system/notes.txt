# Cooldown System

ID: 48_cooldown_system
Level: Level 10 - Mixed Challenges

## Goal
A move can only be used every k turns. Process USE and WAIT commands.

## Expected Format
- Input: First line: command count and cooldown length k. Next lines: USE or WAIT.
- Output: One result line for each command.

## Sample
Input:
```text
5 2
USE
USE
WAIT
USE
USE
```

Output:
```text
Move used
On cooldown
Waiting
Move used
On cooldown
```

## Skills
state tracking, commands

## Extra Detail
Be precise about when cooldown decreases after each turn.
