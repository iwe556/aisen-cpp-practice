# Same Letter Count

ID: 18_same_letter_count
Level: Level 04 - Hash Maps

## Goal
Check whether two words contain the same letters with the same frequencies.

## Expected Format
- Input: Two lines, one word per line.
- Output: Print Same or Not Same.

Output:
