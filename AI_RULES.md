# AI Rules for This Practice Repo

The purpose of this repo is to improve my independent problem-solving.

AI is allowed to:
- Explain C++ syntax
- Explain compiler errors
- Give small hints
- Help create test cases
- Review my solution after I attempt it
- Point out bugs without rewriting the whole solution

AI is not allowed to:
- Solve the problem immediately
- Write the full algorithm before I try
- Replace my thinking with a completed answer
- Auto-fill entire solution files

For every problem, I should first:
1. Read `notes.txt`.
2. Restate the task in my own words.
3. Write 2 custom tests.
4. Attempt `solution.cpp`.
5. Test it.
6. Record the result in the attempt log.
7. Only then ask for help if stuck.
