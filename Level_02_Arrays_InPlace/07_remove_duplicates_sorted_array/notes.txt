# Remove Duplicates from Sorted Array

ID: 07_remove_duplicates_sorted_array
Level: Level 02 - Arrays In-Place

## Goal
Given a sorted array, print each distinct value once without using a set or map.

## Expected Format
- Input: First line: n. Second line: n sorted integers.
- Output: The unique values in sorted order.

## Sample
Input:
```text
7
1 1 2 2 3 4 4
```

Output:
```text
1 2 3 4
```

## Skills
sorted arrays, duplicate handling

## Extra Detail
Because the array is sorted, duplicates are adjacent.
