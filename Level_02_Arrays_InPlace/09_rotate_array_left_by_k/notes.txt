# Rotate Array Left by k

ID: 09_rotate_array_left_by_k
Level: Level 02 - Arrays In-Place

## Goal
Move the array left by k positions, wrapping removed values to the end.

## Expected Format
- Input: First line: n and k. Second line: n integers.
- Output: The rotated array.
