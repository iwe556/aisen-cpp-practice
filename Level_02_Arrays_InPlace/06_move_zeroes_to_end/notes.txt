# Move Zeroes to End

ID: 06_move_zeroes_to_end
Level: Level 02 - Arrays In-Place

## Goal
Move all zeroes to the end while keeping non-zero values in their original order.

## Expected Format
- Input: First line: n. Second line: n integers.
- Output: The transformed array values in order.

## Sample
Input:
```text
6
0 5 0 3 8 0
```

Output:
```text
5 3 8 0 0 0
```

## Skills
arrays, in-place updates

## Extra Detail
Try cases with no zeroes and all zeroes.
