# Find the Missing Number

ID: 05_find_missing_number
Level: Level 01 - Basics

## Goal
The numbers should be from 1 to n, but exactly one number is missing. Read the incomplete list and print the missing value.

## Expected Format
- Input: First line: n, the full intended range 1..n. Second line: n-1 distinct integers from that range.
- Output: One integer: the missing number.

## Sample
Input:
```text
5
1 2 4 5
```

Output:
```text
3
```

## Skills
arrays, arithmetic, careful input size

## Extra Detail
The input count is n-1, not n. The missing number might be 1 or n.
