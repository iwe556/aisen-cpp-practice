# Clean Palindrome

ID: 11_clean_palindrome
Level: Level 03 - Strings

## Goal
Check whether a sentence is a palindrome while ignoring spaces and letter case.

## Expected Format
- Input: One full line of text.
- Output: Print Palindrome or Not Palindrome.

## Sample
Input:
```text
Never odd or even
```

Output:
```text
Palindrome
```

## Skills
strings, case handling, two ends of a string

## Extra Detail
Use getline if the input can contain spaces.

## Before Coding
- Restate the task in one sentence.
- Write 2 custom tests, including one edge case.
- Note any unclear behavior you decide for yourself.
