# Compress String

ID: 12_compress_string
Level: Level 03 - Strings

## Goal
Compress consecutive repeated characters by writing the character followed by its run length.

## Expected Format
- Input: One string.
- Output: The compressed string.

## Sample
Input:
```text
aaabbccccd
```

Output:
```text
a3b2c4d1
```

## Skills
strings, consecutive counting

## Extra Detail
Single characters still have count 1 in the sample style.
