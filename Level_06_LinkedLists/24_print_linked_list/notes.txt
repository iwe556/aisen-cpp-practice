# Print Linked List

ID: 24_print_linked_list
Level: Level 06 - Linked Lists

## Goal
Given a manually created linked list, print each node value from head to tail.

## Expected Format
- Input: Usually no standard input; create the sample list in code if the prompt asks.
- Output: Node values in order.

## Sample
Input:
```text
Example: 10 -> 20 -> 30 -> null
```

Output:
```text
10 20 30
```

## Skills
linked-list traversal

## Extra Detail
Move from one node to the next until there is no next node.
