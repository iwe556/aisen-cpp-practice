# Insert After Value

ID: 28_insert_after_value
Level: Level 06 - Linked Lists

## Goal
Find a node with a target value and insert a new node after it.

## Expected Format
- Input: A linked list, target value, and new value.
- Output: The linked list after insertion.

## Sample
Input:
```text
Example: 10 -> 20 -> 40, insert 30 after 20
```

Output:
```text
10 -> 20 -> 30 -> 40
```

## Skills
pointer relinking

## Extra Detail
Decide what to print if the target value is not found.
