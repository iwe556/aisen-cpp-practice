# Undo System

ID: 22_undo_system
Level: Level 05 - Stack and Queue

## Goal
Process WRITE x and UNDO commands, then print the remaining written values.

## Expected Format
- Input: First line: number of commands. Next lines: WRITE value or UNDO.
- Output: The remaining written values in order.

## Sample
Input:
```text
5
WRITE A
WRITE B
UNDO
WRITE C
WRITE D
```

Output:
```text
A C D
```

## Skills
stack behavior with vector

## Extra Detail
Decide what happens if UNDO is called when nothing is written.

## Before Coding
- Restate the task in one sentence.
- Write 2 custom tests, including one edge case.
- Note any unclear behavior you decide for yourself.
