# Valid Brackets Easy

ID: 21_valid_brackets_easy
Level: Level 05 - Stack and Queue

## Goal
Read a string containing only '(' and ')' and decide whether it is balanced.

## Expected Format
- Input: One bracket string.
- Output: Print Valid or Invalid.

## Sample
Input:
```text
(()())
```

Output:
```text
Valid
```

counter or stack idea

## Extra Detail
A closing bracket cannot be matched by a future opening bracket.
